# Teil

## Commands (`yarn *`)

### `docs:dev`

Starts the `./docs` Nuxt app on [localhost:9991](http://localhost:9991).

### `dev`

Runs `docs:dev` in parallel.

### `test`

Runs the tests.

### `test:watch`

Re-run tests on file changes.

### `lint`

Linting all the code.
